Докер калькулятор.

Создаём новую папку для проекта 
mkdir calculator-app
cd calculator-app

Создаём Dockerfile:
touch Dockerfile
nano Dockerfile

Команда для сборки образа:
sudo docker build -t calculator-app .

Команда для запуска контейнера:
docker run -it calculator-app
